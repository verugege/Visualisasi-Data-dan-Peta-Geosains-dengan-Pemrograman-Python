# Pengenalan PyGMT & Pemetaan Dasar

## Deskripsi

Sesi ini dirancang untuk memberikan pemahaman mendalam kepada peserta mengenai dasar-dasar pemetaan geospasial menggunakan PyGMT, sebuah library Python yang mengintegrasikan kemampuan pemetaan dari Generic Mapping Tools (GMT) ke dalam lingkungan pemrograman modern. Peserta akan mempelajari konsep fundamental seperti sistem koordinat dan referensi spasial (CRS), proyeksi peta, pengaturan region, serta pengelolaan layers dalam pembuatan peta geospasial.

Materi difokuskan pada praktik pembuatan peta tektonik regional, di mana peserta akan belajar menampilkan topografi permukaan menggunakan grdimage, menambahkan elemen penting seperti coastline, batas lempeng (plate boundaries), jalur gunung api (volcanic arc), dan palung samudra (trenches). Selain itu, peserta akan membuat visualisasi sebaran gempa bumi menggunakan data spasial aktual dengan pewarnaan berdasarkan kedalaman (depth) dan ukuran titik yang merepresentasikan magnitudo (magnitude). Melalui interpretasi pola seismisitas yang dihasilkan, peserta akan memahami bagaimana pemetaan berbasis PyGMT dapat digunakan untuk analisis geotektonik secara informatif dan profesional.
